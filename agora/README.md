# OLYMPUS: The Agora
*Community Companion for OpenAI Parameter Golf*
*Powered by Light Speed Up*

## What is this?

The Agora is a community-driven transparency hub for the [OpenAI Parameter Golf](https://github.com/openai/parameter-golf) competition. It provides:

- **Submission Checklist** — plain-English rules, what's legal/banned/grey area
- **Rule Change History** — every ruling with dates and affected PRs
- **Dual Leaderboard** — compliant submissions vs full archive (including banned)
- **Funding Transparency** — who's spending what, grants vs self-funded
- **Blocked on Compute** — researchers who have ideas but no GPU access
- **Compute Survival Guide** — serverless-first, tested by us
- **Technique Map** — what's been tried, what works, what's untried (coming)
- **BPB Timeline** — SOTA over time with key events (coming)

## Why?

The competition has 1000+ PRs and no clear source of truth. Issue #140 has 133KB of data. The March 27 ruling silently killed 30+ submissions. Participants spend real money on illegal approaches.

We build what the organizers haven't: a clear, public resource for the community.

## Conflict of Interest

We are active participants. Our PR #769 was closed in the March 27 ruling. All classifications are automated from public GitHub data and disputable via [Issues](https://github.com/MatoTeziTanka/parameter-golf/issues).
